This plugin allows you to easily customize your WooCommerce shop and apply to EU law for webshops.

== Description ==

This WordPress and WooCommerce plugin makes applying to the new [EU laws](http://eur-lex.europa.eu/legal-content/EN/TXT/?uri=CELEX:32011L0083) easy for every webshop owner.

It allows easy customization of the order button text and gives you the possibility to add additional
notices for return conditions.

= Languages =

This plugin is fully translatable. The admin panel comes by default with Englisch and Dutch translations and the POT files are included for own translation.

The text for the checkout button and checkout screen is customizable trough the WooCommerce settings.


= Regulations =

This plugin has been developed for the new EU regulations that hace taken effect since June 13, 2014.

The new regulation state that buyers in an webshop should get a specific text on the order button e.g. 'Order with payment obligation'


= Future =

The EU regulation goes further then the text on your order betton. They alsoy demand the presents of a default return form by wich consumers
can return their order within 14 days.


== Installation ==

1. Upload the entire `WooCommerce-Extender` plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by selecting `WooCommerce`, `Settings`, `Regulation`

== Frequently Asked Questions ==

= The plugin doesn't work =

Ensure that you have WooCommerce version 2.2 or later running and WordPress version 3.9 or later.
